# Cardápio Online

### 

### Objetivo:

Criar um sistema de cardápio online para facilitar o atendimento a clientes em lanchonetes e hamburguerias, além de oferecer opções de controles de estoques e gerenciamentos dos serviços.

### Levantamento de Requisitos:

1. Objetivo do sistema:
    Sistema responsável por gerenciar os pedidos feitos pelos clientes. 

2. Quem utilizará o sistema? 
    Serão usuários do sistema: Colaboradores (garçom, caixa e cozinha), gerência e os clientes. 

3. O que cada usuário poderá fazer? 
    * Colaboradores: 
    - Garçom: Receber um alerta quando os pedidos estiverem prontos e receber um alerta quando alguma mesa solicitar seus serviços. 
    - Caixa: irá gerenciar os pedidos direcionando para a cozinha escolhendo prioridade. Além de confirmar pagamento antes ou após o consumo. 
    - Cozinha: irá receber os pedidos, sinalizar quando estiverem sendo preparados, sinalizar quando estiverem prontos. 
    - Gerência: Terá acesso a todas as funcionalidades que os outros usuários terão, além de poder acrescentar novos colaboradores ao banco de dados e gerenciamento do banco de dados de produtos, serviços e estoque e poderá alterar preços e inserir promoções. 
    
    * Clientes: Poderão escolher entre os produtos disponíveis no cardápio, pedir os serviço do garçom, efetuar o pagamento diretamente pelo aplicativo ou escolher pagar no caixa.
