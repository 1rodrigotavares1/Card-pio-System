# Cardápio Online

### 

### Objetivo:

Criar um sistema de cardápio online para facilitar o atendimento a clientes em lanchonetes e hamburguerias, além de oferecer opções de controles de estoques e gerenciamentos dos serviços.


### Tecnologias utilizadas:

    Projeto desenvolvido utilizando Java - Spring Boot - MySQL - HTML - CSS - JavaScript - Git - GitHub
